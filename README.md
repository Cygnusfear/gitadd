# ➕💝 gitadd

Happy little CLI tool for quick interactive GIT staging

![Example Image](demo.gif)

# Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

To build:

```bash
bun run build
```

Find the binary in `dist`
